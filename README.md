## Topik

Sistem Rekam Medis Rumah Sakit dengan menggunakan struktur data AVL Tree

Alasan untuk menggunakan AVL Tree adalah **AVL menggunakan O(log n) untuk semua operasi, mendukung range query dan traversal terurut.**

## Backstory

Ada sebuah rumah sakit besar menangani ribuan pasien setiap bulannya yang bernama SUNIB's Hospital. Setiap pasien yang datang dengan kondisi apa pun diberikan satu Nomor ID pasien yang bersifat unik dan permanen. Sistem yang digunakan oleh SUNIB's Hospital adalah data structure BST (Binary Search Tree), dimana data pasien akan disorti berdasarkan nilai nomor ID Pasien. Tetapi, sistem BST kurang efektif dalam pencarian data kalau bentuk tree-nya tidak sempurna, atau satu sisi. Oleh karena itu, SUNIB's Hospital membutuhkan pertolongan dari REnercord (Kelompok kita) untuk membuat sistem BST yang teroptimalkan, yaitu AVL Tree. Dan dengan itu, kita ingin membuat SUNIB's Hospital v2 dimana sistemnya adalah data structure AVL Tree.
